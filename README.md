# ProcureIQ — AI Procurement Intelligence Platform

An enterprise-grade React application for lithium-ion battery procurement management with AI-powered analytics.

## Features

1. **Procurement Dashboard** — 12+ live KPIs with predictive analytics and AI forecasting
2. **AI Report Generator** — Generate, schedule, and export reports with real-time telemetry
3. **Root Cause Analysis** — Interactive tree-based drill-down for KPI anomalies
4. **NLP Intelligence Bot** — Ask historical and predictive questions, get chart-enriched answers

## Quick Start

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack
- React 18
- Recharts (data visualization)
- Lucide React (icons)
- Inter + Noto Sans JP fonts
- Pure CSS (no Tailwind/Bootstrap)

## Design System
- Primary: #2ca6ff
